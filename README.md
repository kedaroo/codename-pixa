## Contributing
1. Fork the repository.
2. Clone your fork:

   ```bash
   git clone https://github.com/kedaroo/codename-pixa.git
   ```
4. Install dependencies:
   
   ```bash
   npm ci
   ```
6. Make your changes and run the app locally:
   
   ```bash
   npm run dev
   ```
